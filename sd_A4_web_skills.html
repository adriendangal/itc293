What I Built for This Portfolio

For Assessment 4 I extended the A3 portfolio by adding JavaScript interactivity and a newsletter signup form for NexaCore Corporation. Key technical decisions included:

  • All JavaScript moved to an external scripts.js file for reusability.
  • Form validation uses a named function validateNewsletterForm() — called via onsubmit on the form element.
  • An onload alert greets users when the newsletter page opens.
  • CSS variables (:root) keep the colour scheme consistent and easy to update.
  • A setActiveNav() function automatically highlights the correct nav button on each page.
