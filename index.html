About This Site

This portfolio is my Assessment 4 submission for ITC293. It builds directly on the A3 submission, extending the site with three additional skills pages, an interactive JavaScript page, and a newsletter signup form for the fictitious company NexaCore Corporation.

The website is built using semantic HTML5, a single external CSS file, and an external JavaScript file for reusable functions. The design is consistent across all pages using shared CSS classes such as card, nav-btn, and skills-grid. Press Ctrl+P or Cmd+P to preview the print version of any page.
