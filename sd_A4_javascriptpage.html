About NexaCore Corporation

NexaCore Corporation is a fictitious technology company used for this assessment. NexaCore specialises in digital transformation services for small and medium businesses, offering solutions in cloud infrastructure, AI integration, cybersecurity, software development, and IT support.

The company values innovation, clear communication, and building lasting relationships with its clients across Australia.

Stay Up to Date — Newsletter Signup

Subscribe to the NexaCore newsletter to receive updates on new services, industry news, and technology tips. The form opens in a new tab for a distraction-free signup experience.

Clicking the button below will open the newsletter signup form in a new browser tab using JavaScript's window.open() function.

JavaScript Features on This Page

This page demonstrates the following JavaScript concepts implemented for Assessment 4:

  • window.open() — opens the newsletter form in a new tab.
  • External JS file — scripts.js contains all functions, keeping HTML clean.
  • setActiveNav() — automatically highlights the correct nav link on each page.
  • validateNewsletterForm() — validates required fields before form submission.
  • showWelcomeAlert() — displays an onload alert on the newsletter page.
